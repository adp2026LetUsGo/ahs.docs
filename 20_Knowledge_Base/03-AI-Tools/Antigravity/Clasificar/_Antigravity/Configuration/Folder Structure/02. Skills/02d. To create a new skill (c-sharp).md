#### 2. Example `SKILL.md` (for .NET 10 Services)

This skill ensures the agent uses modern C# features and adheres to Clean Architecture principles.



``` markdown
---
name: dotnet-service-generator
description: Generates C# service classes and interfaces using .NET 10 conventions. Trigger when the user asks to "add a service," "create a repository," or "scaffold a domain layer."
---

# .NET 10 Service Generator
Follow these rules when generating code:

1. **Modern Syntax**: Use file-scoped namespaces and Primary Constructors for Dependency Injection.
2. **Nullable Reference Types**: Ensure `#nullable enable` is respected and use `?` for optional properties.
3. **Template**: Base the implementation on the pattern in `./resources/service-template.cs`.
4. **Registration**: After creating the file, use `./scripts/register_service.py` to automatically update `Program.cs` with the correct `builder.Services.AddScoped<ISample, Sample>();` line.
5. **Project Context**: Check the current directory to determine if the file belongs in `.Core`, `.Infrastructure`, or `.API`.
```

Use code with caution.

``` csharp
namespace {{NAMESPACE}};

    Task<IEnumerable<string>> GetItemsAsync(CancellationToken ct = default);

public class {{NAME}}Service(ILogger<{{NAME}}Service> logger) : I{{NAME}}Service

    public async Task<IEnumerable<string>> GetItemsAsync(CancellationToken ct = default)
```

#### 4. Integration Script

Since Antigravity IDE agents can execute local scripts, you can use a Python or Bash script to "inject" the new service into your **Minimal API** or `Program.cs`.

- **Script Path**: `.agent/skills/dotnet-service-generator/scripts/register_service.py`
- **Logic**: The script searches for `builder.Services` in your project and inserts the new registration line.

Why this works for .NET 10

- **Automated Boilerplate**: .NET 10 often involves creating an interface + implementation; the agent does both in one command.
- **Version Pinning**: You can explicitly tell the agent in `SKILL.md` to _never_ use older syntax like `public class MyClass { public MyClass(IDi service) { ... } }`.
