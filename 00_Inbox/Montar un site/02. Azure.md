puedo usar mi anconhs.com mi dominio en azure y que me salga gratis... es un pagina sencilla

Sí, puedes usar tu dominio en [Azure](https://azure.microsoft.com/en-us) de forma gratuita si tu página es sencilla (estática), pero debes elegir el servicio correcto, ya que no todos los servicios de Azure permiten dominios personalizados en sus capas gratuitas. 

Para que sea gratis en 2026, la opción recomendada es **Azure Static Web Apps**:

1. El servicio gratuito: Azure Static Web Apps

A diferencia del "App Service" convencional (que cobra por usar dominios personalizados), el plan gratuito de **Static Web Apps** sí incluye soporte para dominios propios sin costo adicional. 

- **Costo:** $0 (Plan Free para proyectos personales).
- **Incluye:** Certificado SSL gratuito (HTTPS) y hasta 2 dominios personalizados por aplicación.
- **Limitación:** Solo sirve para páginas estáticas (HTML, CSS, JS) o frameworks como React, Vue o Angular. No soporta bases de datos tradicionales en el mismo servidor (como PHP/MySQL), aunque puedes usar APIs externas. 

2. Pasos para conectar tu dominio de GoDaddy

Para configurar `anconhs.com` en Azure:

1. **Crea el recurso:** En el portal de Azure, busca e instala una **Static Web App**.
2. **Configura el dominio:** En el menú lateral de tu aplicación en Azure, ve a **Custom domains** y selecciona **Add** > **Custom domain on other DNS**.
3. **Validación en GoDaddy:** Azure te dará un código (registro TXT o CNAME). Debes entrar a tu panel de GoDaddy y añadir ese registro en la sección DNS para demostrar que eres el dueño del dominio.
4. **Certificado SSL:** Una vez validado, Azure generará automáticamente el certificado de seguridad gratis para tu sitio. 

5. Lo que NO es gratis

- **Azure App Service:** Si intentas subir tu web ahí usando el "Plan F1 (Gratis)", verás que la opción de "Custom domains" está bloqueada. Para usar tu dominio ahí, tendrías que subir a un plan de pago (como el B1), que tiene un costo mensual.
- **Renovación del dominio:** Debes seguir pagando la anualidad de `anconhs.com` directamente a GoDaddy; Azure solo te da el hospedaje gratis, no el nombre del dominio. 

Si tu sitio es una página informativa sencilla, **Azure Static Web Apps** es tu mejor opción para mantener el gasto en cero durante 2026.
