### Propuesta de Estructura de Proyectos (GCP)

Usa tus 10 proyectos disponibles para agrupar tus 40 micro-servicios por **Naturaleza Técnica** o **Gran Dominio**:

1. **AHS-Core-Project:** Para el `Shared/`, `Common/`, y `Platform/` del Monorepo. (1 API Key).
    
2. **AHS-Pharma-Vertical:** Para todos los Micro-SaaS de salud/farma (pueden ser 5 o 10 apps aquí). (1 API Key).
    
3. **AHS-Fintech-Vertical:** Para contabilidad, facturación y finanzas. (1 API Key).
    
4. **AHS-Logistics-Vertical:** Para trazabilidad, depósitos y transporte. (1 API Key).
