1. [**Set up hosting**: To make your app available to Shopify and to app users, you need to host it](https://shopify.dev/docs/apps/deployment/web)[1](https://shopify.dev/docs/apps/deployment/web). [You can host your app using any service or platform you prefer](https://shopify.dev/docs/apps/deployment/web)[1](https://shopify.dev/docs/apps/deployment/web). [In the tutorial provided by Shopify, they use Fly.io and Heroku as examples](https://shopify.dev/docs/apps/deployment/web)[1](https://shopify.dev/docs/apps/deployment/web).
    
2. [**Retrieve environment variables**: Apps created using Shopify CLI use environment variables for configuration](https://shopify.dev/docs/apps/deployment/web)[1](https://shopify.dev/docs/apps/deployment/web). [During local development, Shopify CLI provides the environment variables directly to the environment](https://shopify.dev/docs/apps/deployment/web)[1](https://shopify.dev/docs/apps/deployment/web). [However, to deploy your app, you need to retrieve these variables for your app so you can set them in a later step](https://shopify.dev/docs/apps/deployment/web)[1](https://shopify.dev/docs/apps/deployment/web).

[Follow these requirements as you build and prepare to submit your app to ensure your app is published without hiccups](https://shopify.dev/docs/apps/deployment/web)[2](https://www.shopify.com/partners/blog/how-to-build-a-shopify-app).
