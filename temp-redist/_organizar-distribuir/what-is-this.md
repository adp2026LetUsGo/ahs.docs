# AHS.DevKnowledge — ¿Qué es este vault?

Vault local para conocimiento de desarrollo: arquitectura, backend .NET, Azure, DevOps, ML/AI,
datos, frontend, seguridad, proyectos (AHS.FinTechBook, AHS.MicroSaaS) y recursos de aprendizaje.

- Estructura ordenada con prefijos numéricos para navegación rápida.
- En cada carpeta hay un **what-is-this.md** con su propósito.
- Recomendado: proteger esta ruta con Windows Backup / File History.
